# pdu-area-optimizer

This C++ program implements a floorplanning area optimizer based on Lloyd's algorithm and a BFS.

## Building
Follow the installation instructions for [Maplot++](https://alandefreitas.github.io/matplotplusplus/integration/cmake/install-as-a-package-via-cmake/).

Afterwards, using cmake:

```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## Running
```bash
./program [number of iterations]
```
